## 例子目录
这个目录存放使用例子，供您做使用参考。      
目标读者：本 gem 的使用者。   

## 运行例子前请注意
需要先修改根目录下 `.env.example` 文件，提供配置信息。  
建议您新建一个全新的`用户池`，专门用于运行例子，因为例子里会演示"注册,登录"等操作。
